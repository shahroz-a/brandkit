# Governance

BrandKit OS starts as a maintainer-led open-source project.

## Maintainer Responsibilities

- Review changes for determinism, accessibility, and local-first behavior.
- Keep roadmap decisions public.
- Label approachable work for new contributors.
- Protect contributor safety and project quality.

## Decision Making

Small changes can land through pull request review. Large changes should start with an issue or discussion that covers motivation, design, compatibility, and migration impact.
